Clustering - Learn to assign labels

K-Clustering :

  • Procedure:
  1. Choose K - the number of clusters.
  2. Randomly put k feature vectors, which is called centroids, to the feature space.
  3. Compute the distance from each example x to each centroid c using some metric, like the Euclidean distance.
  4. Assign the closest centroid to each example.
  5. Calculate the average feature vector of the examples labeled with it for each centroid, and these average feature vectors are new locations of the centroids.
  6. Recompute the distance from each example to each centroid, modify the assignment. Repeat the procedure until the assignments don’t change after the centroid locations were recomputed.
  • The initial position of centroids influence the final positions. The results of to runs can result in different models.
  • Unspecified K:
  • The elbow method
  • plots the value of the cost function produced by different K.
  • As K increases:
  • The average distortion will decrease.
  •  Each cluster will have fewer constituent instances
  • the instances will be closer to their respective centroids.
  • As K increases:
  • the improvements to the average distortion will decline
  • The value of K at which the improvement to the distortion declines the most is called the elbow.
  • Evaluation:
  1. Measuring the distortion
  2.  the silhouette coefficient

  1. a measure of the compactness and separation of the clusters
  2. increases as the quality of the clusters increase
  3. large for compact clusters that are far from each other
  4. small for large and overlapping clusters

PCA - Principal Component Analysis

  • What Does It Do?
  • Reduces a set of possibly-correlated, high-dimensional variables to a lower-dimensional set of linearly uncorrelated synthetic variables called principal components.
  • Reduces the dimensions of a data set by projecting the data onto a lower-dimensional subspace
  • Analogy:
  • Describing a 3D object in 2D:

  • Rationale: Eigenfunction and eigenvalues.
  • Procedure:
  1. subtract the mean of each explanatory variable from each observation
  2. calculate the principal components of the data, which are eigenvectors of the data's covariance matrix ordered by their eigenvalues

  1. Project data onto the principal components
